A Python toolkit for sound source separation.

## Installation
You can install by pip.
```shell
pip install git+https://github.com/tky823/ssspy.git
```
or clone this repository.
```shell
git clone https://github.com/tky823/ssspy.git
cd ssspy
pip install -e .
```
